Welcome to the Timelapse Project!

The Timelapse Project is the open source hardware and software project which is made to create dynamic time-lapse videos.

Using this instructions you should be able to make the device yourself. If after reading the instructions something remains unclear please ask, I will be happy to answer or correct this instructions.

To make the device you would need:

  1. WeMos D1 or WeMos D1 mini controller
  2. A stepper motor driver board
  3. A stepper motor
  4. A ballhead camera mount
  5. A rigid box
  6. Tools to make a hole in the box and mount the stepper motor inside the box
  7. Wires
  8. 2 leds of different colours
  9. An infra red led
  10. Resistors to limit current which is going through leds
  11. If you buy the controller boards without pre soldered pins you would need soldering gun, solder and alcohol (or something else to clean the soldering parts).

All this stuff could be bought from Chinese web stores and delivered to you by post service.

Basically all you need to do to create a hardware part of the device is to connect pins using wires. In the WeMos board you will find a number of pins. Most of the pins are programmable and have a positive polarity. These are pins which are using in the project:

  • D7. - IR led for controlling the camera
  • D1. - The led which indicates that the wifi communication was established between the device and the wifi point;
  • D6. - The led which indicates that the power is on and the software is working;
  • D2, D4, D3, D5. - Pins to controll the stepper motor. This pins should be connected to the stepper motor driver board.

All leds should be connected to pins via resistor to limit current. To calculate a resistor value Ohm law could be used. Let's say we have a led and it can handle 20 mA current. Then:

by the Ohm law V = I * R

that means R = V / I

3.5 / (20 / 1000) = 175 ohm

It is not nessesary to use a resistor with the exact value. But the value should not be lower then calculated value because the lifetime of led will be reduced. And the value shold not be much higher because the led will be glowing dimmer.

Also mind the polarity. Ground pins of the WeMos controller should be connected to the leds minus (cathode), and pins D1, D6 and D7 to the leds plus (anode).

Most probably the way of connecting stepper motor driver board will be very straightforward. If not check the information about your driver in the internet.

  1. It should have 4 input pins - this inputs should be connected to the pins D2, D4, D3, D5 of WeMos D1.
  2. It should have 2 DC power supply inputs
  3. It should have 4 output contacts to be connected to the stepper motor.
